Challenges / Utmaningar

  • 🌍 Can you connect to a webpage that activates in the map Africa, Russia, or Sweden?
  • 🟠 Can you find a website that shows only one red dot?
  • 🔎 Find a website that activates as many dots as possible!
  • 🎵 Find a website with as much sound as possible!
  • 🤔 Visit your favorite website. How many other websites do you visit at the same time?
  • 🌍 Kan du ansluta till en webbsida som aktiveras på kartan Afrika, Ryssland eller Sverige?
  • 🟠 Kan du hitta en webbsida som bara visar en enda röd punkt?
  • 🔎 Hitta en webbplats med så många punkter som möjligt!
  • 🎵 Hitta en webbplats med så mycket ljud som möjligt!
  • 🤔 Besök din favoritwebbplats. Hur många andra sidor besöker du på samma gång?
This installation is named after Nicola Pellow. She is an Internet pionner who programmed the second web browser ever created, in 1991: the Line Mode Browser. More about Pellow
